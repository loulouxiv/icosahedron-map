● The -r/--resolution parameter controls the detail level of country boundary data from https://www.naturalearthdata.com/:

  | Value | Detail           | File Size | Use Case                      |
  |-------|------------------|-----------|-------------------------------|
  | 110m  | Low              | ~800KB    | Quick previews, small prints  |
  | 50m   | Medium (default) | ~5MB      | General use, good balance     |
  | 10m   | High             | ~60MB     | Large prints, fine coastlines |

  The number refers to map scale (1:110 million, 1:50 million, 1:10 million). Higher resolution means more polygon vertices, resulting in smoother coastlines but larger SVG/PDF files and longer processing time.

  # Fast preview
  python -m icosahedron_map -r 110m --pdf

  # High detail for printing
  python -m icosahedron_map -r 10m --pdf
